  ~~~
  # DynamicOutputComponents
  ~~~~~
  ~~~
  DynamicOutputComponents can be used to configure output for
  an App.
  ~~~~~
  ~~~
  A DynamicOutputComponent's output is generated from a file located 
  in the App's `DynamicOutput` directory.
  ~~~~~
  ~~~
  Note: If the file that generates the DynamicOutputComponent's
  output has the extension `.php`, it will be interpreted by roady
  as an executable `php` file, otherwise roady will interpret the
  file as plain text.
  ~~~~~
  ~~~
  `rig --new-dynamic-output-component` can be used to configure a
  DynamicOutputComponent for an existing App.
  ~~~~~
  ~~~
  For example:
  ~~~~~
  ~~~
  ```
  rig --new-dynamic-output-component \
      --for-app Foo \
      --name Bar \
      --initial-output Baz
  ```
  ~~~~~
  ~~~
  Note: More information about `rig --new-dynamic-output-component`
  can be obtained via:
  ~~~~~
  ~~~
  ```
  rig --help new-dynamic-output-component
  ```
  ~~~~~
