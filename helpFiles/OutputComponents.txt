  ~~~
  # OutputComponents
  ~~~~~
  ~~~
  OutputComponents can be used to configure output for an App.
  ~~~~~
  ~~~
  A OutputComponent's output is defined in a OutputComponent
  configuration file in the App's `OutputComponents` directory.
  ~~~~~
  ~~~
  `rig --new-output-component` can be used to configure an
  OutputComponent for an existing App.
  ~~~~~
  ~~~
  For example:
  ~~~~~
  ~~~
  ```
  rig --new-output-component \
      --for-app Foo \
      --name Bar \
      --output Baz
  ```
  ~~~~~
  ~~~
  Note: More information about `rig --new-output-component` can
  be obtained via:
  ~~~~~
  ~~~
  ```
  rig --help new-output-component
  ```
  ~~~~~
