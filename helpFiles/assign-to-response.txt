  ~~~
  ```rig --assign-to-response --for-app --response 
         [--dynamic-output-components] [--output-components] 
         [--requests]
  ```
  ~~~~~
  ~~~
  ### Description:
  ~~~~~
  ~~~
  Assign one or more Requests, OutputComponents, or 
  DynamicOutputComponents to an existing Response or 
  GlobalResponse.
  ~~~~~
  ~~~
  Note: The Requests, OutputComponents, and 
  DynamicOutputComponents must be configured 
  by the same App as the Response or GlobalResponse.
  ~~~~~
  ~~~
  ### Flags:
  ~~~~~
  ~~~```--for-app```~~~~~                    
                               ~~~
                               The name of the App that 
                               configured the target Response 
                               or GlobalResponse.
                               ~~~~~
                               ~~~
                               WARNING: The App must exist at 
                               the path that corresponds to the 
                               path assigned to the 
                               `--path-to-apps-directory` flag.
                               ~~~~~
                               ~~~
                               Note: For more information about 
                               the `--path-to-apps-directory` 
                               flag use: 
                               ~~~~~
                               ~~~
                               ```
                               rig --help path-to-apps-directory
                               ```
                               ~~~~~
  ~~~```--response```~~~~~                   
                               ~~~
                               The name of the target Response or 
                               GlobalResponse. 
                               ~~~~~
                               ~~~
                               Note: The Response or 
                               GlobalResponse must already be 
                               configured by the specified App. 
                               ~~~~~
  ~~~```--dynamic-output-components```~~~~~  
                               ~~~
                               The names of the 
                               DynamicOutputComponents to assign 
                               to the Response or GlobalResponse.
                               ~~~~~
                               ~~~
                               Note: The DynamicOutputComponents
                               must already be configured by the 
                               specified App.
                               ~~~~~
  ~~~```--output-components```~~~~~          
                               ~~~
                               The names of the OutputComponents 
                               to assign to the Response or 
                               GlobalResponse. 
                               ~~~~~
                               ~~~
                               Note: The OutputComponents must 
                               already be configured by the 
                               specified App. 
                               ~~~~~
  ~~~```--requests```~~~~~                   
                               ~~~
                               The names of the Requests to 
                               assign to the Response or 
                               GlobalResponse.
                               ~~~~~
                               ~~~
                               Note: The Requests must already 
                               be configured by the specified App.
                               ~~~~~
  ~~~
  ### Examples:
  ~~~~~
  ~~~
  ```
  rig --assign-to-response \
      --for-app Foo \
      --response Bar \
      --dynamic-output-components Baz FooBar \
      --output-components FooBaz  FooBarBaz \
      --requests BarBaz BarFoo
  ```
  ~~~~~
