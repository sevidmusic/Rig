  ~~~
  # Dynamic Output
  ~~~~~
  ~~~
  Apps can configure output that is generated from a file
  in the App's `DynamicOutput` directory.
  ~~~~~
  ~~~
  These files can be php files, html files, plaintext files, or
  any other type of file.
  ~~~~~
  ~~~
  php files will be interpreted as php, all other files will
  be interpreted as plain text.
  ~~~~~
