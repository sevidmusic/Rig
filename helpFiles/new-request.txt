ddms --new-request --for-app --name [--relative-url]

Description:

Configure a new Request for the specified App.

Flags:

--for-app                    The name of the App to configure the Request for.

                             WARNING: The specified App must exist, if it does
                             not, the new Request will not be configured.

--name                       A name to assign to the new Request.

                             WARNING: The name must be unique, if a Request
                             already exists with the same name the new Request
                             will not be configured.

--relative-url               A relative url to assign to the new Request.

                             For example:

                                 '?request=HelloWorld'

                             Defaults to relative root url:

                                 '/'

Examples:

ddms --new-request --for-app HelloWorld --name HelloRootRequest

ddms --new-request --for-app HelloWorld --name HelloCosmos --relative-url "?request=HelloCosmos"
